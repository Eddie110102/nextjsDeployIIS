## 前端工程師負責部分
### 專案打包指令
1. 輸入 「 npm run build 」，會在資料夾中產出 「 .next/ 」資料夾。
2. 將「 .next/ 」、「 node_modules/ 」、「 public/ 」、「 .env 」、「 package.json 」、「 package-lock.json 」放入伺服器中的專案資料夾。

## 後端工程師負責部分
### 安裝IIS步驟
1. 確認電腦為 windows 專業版
2. 【 控制台 】->【 程式集 】->【 程式和功能 】->【 開啟或關閉windows功能】
